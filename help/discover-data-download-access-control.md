# [!UICONTROL Découvrir le téléchargement des données] Contrôle d’accès {#discover-data-download-access-control}

Le contrôle [!UICONTROL Discover Data Download] permet aux administrateurs [!DNL Marketo Measure] de définir les politiques de téléchargement des données pour les tableaux de bord Discover en fonction des rôles des utilisateurs. Le contrôle couvre toutes les actions de téléchargement de données sur les tableaux de bord Discover.

1. Cliquez sur **[!UICONTROL Accès aux données]** sous [!UICONTROL Sécurité].

   ![1. Cliquez sur Accès aux données sous Sécurité.](assets/compliance-resources-1.png)

1. Cliquez sur la liste déroulante et sélectionnez l’option appropriée pour votre console.

   ![1. Cliquez sur la liste déroulante et sélectionnez l’option appropriée pour votre console.](assets/discover-control-2.png)

| Option | Accès en téléchargement |
| --- | --- |
| Tous les utilisateurs | Tous les utilisateurs peuvent télécharger des données, y compris aux formats PDF et CSV. |
| Admin Users Only | Seuls les utilisateurs administrateurs peuvent télécharger des données, y compris les formats PDF et CSV. |
| Aucune | Personne ne peut télécharger de données, y compris les formats PDF et CSV. |

1. Cliquez sur **[!UICONTROL Enregistrer]** lorsque vous avez terminé.

   ![1. Cliquez sur Enregistrer lorsque vous avez terminé.](assets/discover-control-3.png)

Le paramètre peut ne pas prendre effet tant que les utilisateurs ne se sont pas déconnectés puis reconnectés.
