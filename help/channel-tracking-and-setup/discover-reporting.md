# Rapports Discover {#discover-reporting}

Dans [!UICONTROL Paramètres utilisateur], les utilisateurs auront la possibilité de remplacer la devise par défaut par l’une des devises d’entreprise répertoriées. Cela modifie l’affichage de montants tels que Coût, Revenu et Revenu du pipeline, par exemple, à côté du code de devise à trois lettres.
