# Masquage des types de rapports superflus {#hiding-unnecessary-report-types}

Une fois l’installation terminée et que vous commencez à utiliser les rapports , votre entreprise n’utilisera pas tous les rapports fournis avec le package [!DNL Marketo Measure]. Par conséquent, il est utile de masquer les types de rapports que vous n’utilisez pas afin d’éliminer toute confusion et de permettre un aspect plus net. Vous pouvez masquer tous les rapports de votre choix, mais les rapports identifiés dans l’image ci-dessous sont généralement masqués.

1. Accédez à l’onglet **[!UICONTROL Rapports]**.

1. Cliquez sur le bouton **[!UICONTROL Créer un rapport]** en haut de l’écran.

1. Saisissez le mot « point de contact » pour renseigner les rapports.

1. Cochez la case **[!UICONTROL Sélectionner les types de rapports à masquer]** en haut à gauche.

1. Cliquez sur les rapports marqués d’un X orange pour que votre liste de rapports ressemble à l’image ci-dessous.
